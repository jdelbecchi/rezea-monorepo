# Instructions pour les assistants IA (Antigravity)

## Lancement du projet Rezea
- Pour démarrer l'environnement de développement, utiliser la commande rapide sans `--build` :
  ```powershell
  docker compose -f infra/docker-compose.dev.yml up -d
  ```
- N'utiliser l'option `--build` que si les configurations Docker (`Dockerfile`, `docker-compose.dev.yml`) ou les dépendances (`requirements.txt`, `package.json`) ont été modifiées.
